Basics of the Web

A person will log into a computer and select a browser such as Firefox, Chrome, or IE. The browser will be used to navigate around the world wide web/internet. The navigation is achieved by placing an HTTP (Hypertext Transfer Protocol ) request via the browser to the servers that contain all of the web data. The request must be entered in a language that the computer will understand and that is know as the HTML (Hypertext Markup Language). If the HTML is properly coded, the request will be fulfilled and the recipient will receive the information back on their computer.



HTML is the basis for everything on the web. It includes the text which is what you see. The markup drives the format which includes references, images, and links to other pages. HTML coding is also referred to as elements. The elements are composed of opening tags and closing tags with the content between the two tags. Unless it is a void tag which does not require a closing tag such as a break or image tag. Tags can be used to make items bold or italicized. Also, tags can be used to link to other websites or to add images. The elements are either inline or block. Block elements make an invisible box around the text.
